# Проект UrFu C++23 & SFML 3.1.0

Современный каркас проекта на C++23 с использованием графической библиотеки SFML 3.1.0.

---

## 🛠 Требования к окружению

Для сборки проекта на Ubuntu/Linux требуются следующие компоненты:

1. **Компилятор C++:** `GCC 14` (пакет `g++-14`).
2. **Система сборки:** `CMake` (версия 3.20+) и утилита `make`.
3. **SFML 3.1.0:** Установленная локально в директорию `~/.local` (включая зависимости).
4. **Дополнительные системные библиотеки:**
   ```bash
   sudo apt install libmbedtls-dev libssh2-1-dev
   ```

## 📂 Структура проекта

```text
├── CMakeLists.txt      # Конфигурационный файл CMake для сборки проекта
├── Makefile            # Удобная обертка для команд CMake (сборка, запуск, очистка)
├── README.md           # Этот файл с инструкциями
├── include/            # Директория для заголовочных файлов (*.hpp)
│   └── Game.hpp        # Интерфейс класса Game
└── src/                # Директория для исходного кода (*.cpp)
    ├── Game.cpp        # Реализация класса Game (игровой цикл, обработка событий, рендеринг)
    └── main.cpp        # Точка входа в программу
```

## 🚀 Сборка и запуск

Все операции сборки обернуты в удобный `Makefile`:

* **Собрать проект:**
  ```bash
  make
  ```
  *(команда сгенерирует файлы CMake и скомпилирует приложение в папку `build/`)*

* **Запустить приложение:**
  ```bash
  make run
  ```
  *(собирает измененные файлы и запускает окно приложения)*

* **Очистить артефакты сборки:**
  ```bash
  make clean
  ```
  *(полностью удаляет папку `build/`)*

* **Пересобрать проект с нуля:**
  ```bash
  make rebuild
  ```

## 💡 Особенности SFML 3.1.0 в этом проекте

* **События (Events):** В SFML 3 метод `pollEvent()` возвращает `std::optional<sf::Event>`. Обработка нажатий клавиш производится безопасным способом через метод `.getIf<T>()`:
  ```cpp
  while (const std::optional event = window.pollEvent()) {
      if (event->is<sf::Event::Closed>()) {
          window.close();
      }
      else if (const auto* keyPressed = event->getIf<sf::Event::KeyPressed>()) {
          if (keyPressed->code == sf::Keyboard::Key::Escape) {
              window.close();
          }
      }
  }
  ```
* **Инициализация окна:** Конструктор `sf::VideoMode` теперь принимает размеры окна в виде списка инициализации: `sf::VideoMode({800, 600})`.
* **CMake-таргеты:** Библиотеки линкуются с использованием современного пространства имен (например, `SFML::Graphics`).
